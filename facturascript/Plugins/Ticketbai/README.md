# Ticketbai
Implementación de TicketBai para FacturaScripts.
- https://facturascripts.com/plugins/ticketbai
- https://tbai.barnetik.com/api

## Requisitos
Requiere PHP 7.4 o superior con la extensión gmp instalada y activada.

## Licencia
ESTE PLUGIN NO ES SOFTWARE LIBRE. NO SE PERMITE SU DISTRIBUCIÓN SIN AUTORIZACIÓN.

## Nombre de carpeta
Como con todos los plugins, la carpeta se debe llamar igual que el plugin. En este caso **Ticketbai**.

## Datos de configuración
- La empresa debe tener dirección del país vasco y un NIF válido.
- El cliente debe tener un nif válido.
