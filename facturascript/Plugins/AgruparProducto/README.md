# FacturaScripts
Software de código abierto de facturación y contabilidad para pequeñas y medianas empresas.
Software ERP de código abierto. Construido sobre PHP, utilizando de componentes Symfony y Bootstrap 4.
Fácil y potente.

# AgruparProducto
Plugin para FacturaScripts que permite crear diferentes agrupaciones correspondientes
a varias unidades de un mismo producto.

A estas agrupaciones se les pueden asignar descuentos especiales, así como un código EAN personalizado.

Por ejemplo:
Puede crear para el producto PILA
  - una agrupación denominada PACK que se compone de 3 unidades.
  - una agrupación denominada CAJA que se compone de 30 unidades.


<strong>ESTE PLUGIN NO ES SOFTWARE LIBRE. NO SE PERMITE SU LIBRE DISTRIBUCIÓN.</strong>

## Nombre de carpeta
Como con todos los plugins, la carpeta se debe llamar igual que el plugin. En este caso **AgruparProducto**.
