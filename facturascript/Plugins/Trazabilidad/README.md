# Trazabilidad
Añade soporte para números de serie/lote en los productos.

## Documentación
- https://facturascripts.com/plugins/trazabilidad

## Nombre de carpeta
Como con todos los plugins, la carpeta se debe llamar igual que el plugin. En este caso **Trazabilidad**.

## Funcionalidades
- Podemos marcar qué productos tienen trazabilidad.
- Los productos que no controlan stock o permiten vender sin stock, no se pueden marcar como trazables.
- Si el producto tiene trazabilidad, podemos añadir números de serie/lote.
